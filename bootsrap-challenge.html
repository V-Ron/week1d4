Top 10 Web Languages of 2017

by Kari Ann And Veronica

Previous Next

RUBY

Ruby is a programing language created in the mid-a990s by Mastumoto in Japan. Its a dynamic, reflective, obect oriented, general purpose language.

PYTHON

python is an interpretive language emphasizing code readability, and a syntax that allows programmers to express concepts in fewer lines of code.

JAVA

Java is a general purpose language that is class-based, object-oriented and specifically designed to have as few implementation dependencies as possible.

JAVASCRIPT

Javascript is influenced by Java but syntax is more similar to C and is based on capital ECMAscript.

SQL

SQL consists of a data-definition language, data-manipulation language, and data control language. includes data-insterts, query, update, and delete, schema-creation and modification, and data-access control.

REACT

React is a Javascript library for building user interfaces. Allows developers to create large web-application that use data and can change over time without reloading the page.

RUBY ON RAILS

Ruby on rails uses the MVC pattern to organize application programming. The framework mapps to a table in a database and to a Ruby file.

TYPE SCRIPT

Typescript is a free and open-sourced language by Microsoft. its a strict syntactical superset of javacript and adds optional statis typing to the language.

R

R is for statistical computing and graphics and widely used among statisticians and data-miners for developing statistical software and data analysis.

SWIFT

Swift is a general purpose multi0-paradign compiled language by Apple. It's designed to work with apple's COCOA and COCOA touch-frameworks and the large body of existing code written for Apple products.
